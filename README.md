# Vidyanta Tech

Welcome to the Vidyanta Tech codebase! This project is a modern, full-service digital agency website built using React, Vite, Tailwind CSS, and Framer Motion.

## Features

- **Modern UI/UX**: Clean, responsive layout with an engaging and highly-animated user experience.
- **Scroll Animations**: Dynamic and fluid section-by-section scroll transitions utilizing `framer-motion`'s `useInView`. 
- **Sticky Cards**: Custom scrolling behavior in the Services section for a unique stacking card effect.
- **Fully Responsive**: Designed to look great on desktop, tablet, and mobile devices.
- **Micro-interactions**: Subtle hover states, animated buttons, and floating particles scattered throughout to create a lively and polished feel.

## Tech Stack

- **Framework**: React with Vite
- **Styling**: Tailwind CSS & plain CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need [Node.js](https://nodejs.org/) installed on your local machine.

### Installation

1. Install NPM packages:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```
3. Build for production:
   ```sh
   npm run build
   ```
